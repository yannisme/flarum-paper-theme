A [Flarum](http://flarum.org) extension. just a themem

## Installation

Install with composer:

```sh
composer require yannisme/papertheme:"*"
```

## Updating

```sh
composer update yannisme/papertheme:"*"
php flarum migrate
php flarum cache:clear
```
